# Task Caretaker

Task Caretaker is a modern **task management and productivity web application** designed to help users organize daily activities efficiently. The application provides a clean interface for managing tasks, tracking progress, and improving productivity.

This project was built as part of my hands-on learning in modern frontend development using **React and Vite**, with authentication and state management features.

---

## 🚀 Live Demo

https://task-caretaker.vercel.app

---

## ✨ Features

* User authentication (Signup / Login)
* Create and manage tasks
* Mark tasks as completed
* Dashboard interface for task overview
* Dark mode support
* Clean and responsive UI
* Secure environment variable configuration

## ⚙️ Installation & Setup

1. Clone the repository

```
git clone https://github.com/mishraji018/task_caretaker.git
```

2. Navigate to the project folder

```
cd task_caretaker
```

3. Install dependencies

```
npm install
```

4. Create a `.env` file and configure environment variables

```
VITE_API_KEY=your_api_key
VITE_AUTH_DOMAIN=your_auth_domain
VITE_PROJECT_ID=your_project_id
```

5. Start the development server

```
npm run dev
```

6. Open in browser

```
http://localhost:5173
```
