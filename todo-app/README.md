# Task Caretaker – Todo App

Task Caretaker is a simple and efficient Todo application built to manage daily tasks in an organized way.
I developed this project while learning modern web development concepts and improving my understanding of JavaScript and frontend tools.

The application allows users to create, manage, and track tasks easily with a clean and responsive interface.

---

## 🚀 Live Demo

🔗 https://task_caretaker.vercel.app

---

## ✨ Features

* Add new tasks
* Mark tasks as completed
* Delete tasks
* Clean and responsive user interface
* Fast development environment using Vite

## ⚙️ Installation & Setup

1. Clone the repository

git clone https://github.com/mishraji018/task_caretaker.git

2. Navigate to the project directory

cd task_caretaker/todo-app

3. Install dependencies

npm install

4. Run the development server

npm run dev

5. Open the application in your browser

http://localhost:5173
